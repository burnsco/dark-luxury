# Dark Luxury

Dark Luxury is a quiet-luxury storefront concept built with Astro and React. It mixes product browsing, editorial content, and a restrained high-end visual system.

## Highlights

- Multi-page storefront with shop, product, journal, and about pages
- Curated product and category data for a fictional maison
- Dark editorial styling with lightweight client behavior

## Stack

- Astro
- React
- TypeScript

## Quick start

```bash
bun install
bun run dev
```

Checks and build:

```bash
bun run check
bun run build
```
